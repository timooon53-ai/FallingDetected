# **Система обнаружения падений: MediaPipe vs YOLO**


Проект реализует систему обнаружения падений с использованием двух подходов компьютерного зрения: MediaPipe Pose и YOLO v8 Pose.

Система анализирует видео, фиксирует моменты падения и подъёма человека, сохраняет логи и визуальные доказательства.
Возможности
Обнаружение падений в реальном времени двумя независимыми моделями.
Визуализация ключевых точек тела.
Логика обнаружения падений на основе углов наклона туловища.
Ведение логов с отметками времени и длительностью событий.
Хранение данных в базе SQLite.
Пофреймовая запись видео с аннотациями.
Параллельное сравнение результатов обеих моделей.
Как это работает
Система анализирует угол наклона туловища для определения состояния человека:
Стояние: угол близок к вертикальному (в пределах порога).
Падение/лёжа: угол близок к горизонтальному (в пределах порога).
Переход: фиксируется при смене состояния «стояние → падение» или «падение → стояние».
Два независимых конвейера работают параллельно:
MediaPipe Pose — лёгкое решение для оценки позы от Google.
YOLO v8 Pose — модель Ultralytics для обнаружения объектов и поз в реальном времени.
